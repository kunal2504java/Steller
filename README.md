# Steller

Candela + Probatum — Stellar Build Station project.

- **Candela** (`candela-kit`): open-source passkey onboarding UX kit for Stellar — your fingerprint is your wallet.
- **Probatum**: verifiable certificates for the long tail of issuers, sealed on Stellar mainnet.
